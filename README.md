# 🪙 Bitcoin Price Tracker

Um aplicativo desktop simples desenvolvido em **Python** que monitora a cotação do Bitcoin em tempo real para diversas moedas (USD, EUR, BRL e AOA).

## 🚀 Funcionalidades
- **Consumo de API:** Busca dados em tempo real através da API CryptoCompare.
- **Interface Gráfica:** Desenvolvida com a biblioteca `Tkinter`.
- **Multimoedas:** Exibição simultânea de valores em Dólar, Euro, Real e Kwanza.
- **Auto-update:** Os valores são atualizados automaticamente a cada 1 segundo.

## 🛠️ Tecnologias Utilizadas
- [Python](https://www.python.org/)
- [Tkinter](https://docs.python.org/3/library/tkinter.html) (Interface Gráfica)
- [Requests](https://pypi.org/project/requests/) (Consumo de API)
- [Pillow (PIL)](https://python-pillow.org/) (Processamento de Imagens)
